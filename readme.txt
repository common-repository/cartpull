HookMax’s abandoned cart recovery plugin helps your WooCommerce store to recover lost revenue due to cart abandonment.

== Description ==

HookMax’s [WooCommerce abandoned cart](https://hookmax.com) recovery send automated reminder emails to your customers who have added the product to their cart but did not complete the order. 

= Features =

* Easy to configure abandoned cart recovery system.
* You can offload data to hookmax and track insight when you want it. 
* Easily you can edit or add campaigns. 
* Your website will not get big data with abandoned cart tracking and email campaigns. 
* Add or edit or delete campaigns very easily. 
* No need any coding skill. 

= How to Configure =
Go to wp-admin > hookmax Option > press on "connect to hookmax" button then verify.

More details : [HookMax.com](https://hookmax.com/)

== Installation ==

1. Download and extract plugin files to a wp-content/plugin directory.
2. Activate the plugin through the WordPress admin interface.
3. Go to wp-admin > hookmax Option > press on "connect to hookmax" button then verify.
4. Done !
